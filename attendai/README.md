# AttendAI Frontend

Frontend for the AttendAI Prototype.

Live website:

- https://attend-ai-prototype.vercel.app/

## Local development

```bash
npm install
cp .env.example .env
npm run dev
```

Set the frontend backend URL in `.env`:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

For full project documentation, use the repository root README.
